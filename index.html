ROS 2 Differential-Drive Robot for Indoor Mapping and Autonomous Navigation

Designed and integrated a differential-drive mobile robot for indoor mapping, localization, obstacle avoidance, and autonomous point-to-point navigation. The system combines 2D LiDAR readings, wheel odometry, SLAM, path planning, navigation, and embedded motor control.

Technologies: ROS 2 Humble, Raspberry Pi, ESP32, 2D LiDAR, SLAM Toolbox, Nav2, odometry, C, and Python.

Metaheuristic Optimization for PCB Drilling Path Planning

Modeled the PCB drilling sequence as a Traveling Salesman Problem and developed a modular Python implementation to compare a Nearest Neighbor heuristic against a Genetic Algorithm. The evolutionary approach uses tournament selection, ordered crossover, swap mutation, and elitism to explore shorter drilling routes.

Technologies: Python, genetic algorithms, combinatorial optimization, heuristics, Matplotlib, CSV data processing, and parameter tuning.

BiblioSearcher

Developed a modular Python desktop application for collecting, normalizing, and enriching bibliographic information from DOI identifiers and publication URLs. The application retrieves metadata from external APIs, matches journals using ISSN and fuzzy title comparison, and generates structured multi-sheet Excel reports.

Technologies: Python, Tkinter, Pandas, Requests, Crossref API, Elsevier API, RapidFuzz, SCImago datasets, and Excel.

Simplified 8-bit Microprocessor in VHDL

Designed and simulated a modular 8-bit accumulator-based microprocessor using VHDL. The architecture integrates an arithmetic logic unit, reusable registers, a shared bidirectional data bus, 256 × 8-bit RAM, and a finite state machine control unit supporting memory, arithmetic, logic, jump, and no-operation instructions.

Technologies: VHDL-2008, RTL design, finite state machines, digital logic, GHDL, EDA Playground, EPWave, and functional testbenches.
